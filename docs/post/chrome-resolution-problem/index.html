Chrome 分辨率突然变的很低

yuanbo · 2017-04-10

晚上10：00洗漱完之后，回书房正打算写一些东西，Mac电脑突然死机了。重启后其他没有什么问题，打开 Chrome 浏览器，发现辨率很低，字基本上不清楚，而其他应用都没有这个问题。

网上搜索了一下，有人说是字体问题，我试着修改了字体，结果发现没用。有人说需要重置浏览器设置，这个比较麻烦，安装的很多插件都没有了，不过也没办法，只好重置了 Chrome 浏览器，结果发现还是没有用。想了想既然都重置了，也就不在乎下载最新的 Chrome 浏览器重新安装，这样总该没有问题了吧。奇怪的是，重新安装了最新版的 Chrome 浏览器，还是没有用。这下头大了，总不能以后不用 Chrome 了吧，这可是我最爱的浏览器啊。

虽然着急，但也不知道该怎么办了，只好回到应用程序中「command + i」查看下 Chrome，这一查把我给气的啊，也不知道怎么回事，Chrome 浏览器属性里「以低分辨率打开」这个选项，不知道什么时候被勾上了。NND，我说怎么回事呢？把这个设置改回去之后，Chrome 顿时恢复了之前的清晰状态。

你说这叫什么事啊，明天还要重新安装那些个插件，郁闷！以后点上碰见什么问题，还是先自己查明情况，再重置吧。
